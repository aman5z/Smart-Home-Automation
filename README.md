# 🏠 DIY Smart Home & IoT Projects

A collection of my DIY smart home automations and IoT projects built using ESP8266, ESP32, Arduino, and various smart integrations like Alexa, Google Assistant, and Sinric Pro.

---

## 📌 Room #1 – Advanced Smart Bedroom Setup

Back in **2021**, I built a complete automation system using:

* 8-channel relay
* NodeMCU ESP8266
* AC to DC 5V 2A adapter

### 🔌 Connected Devices

* Tube lights
* Ceiling fan
* Philips Wiz Smart LED Bulb (10W)
* DIY RGB Lamp (ESP8266 + RGB strip)
* Study lamp
* 3 × 2-pin smart sockets

### ⚙️ Features

* Auto power-off sockets when devices reach **100% charge**
* Safe overnight charging (prevents overcharging)
* Scheduled automation for power saving

### ❄️ AC Automation

* Controlled via Homemate Wi-Fi IR remote
* Voice control (Alexa & Google Assistant)
* Automation schedule:

  * 24°C until 3 AM
  * 26°C until 5 AM
  * OFF after 5 AM

### 🌬️ Fan Automation

* Turns OFF at **6 AM** (acts as a wake-up trigger)

### 📱 Smart Display (Recycled Device)

* Repurposed **POCO M3**
* Removed ~90% bloatware
* Custom holder with:

  * 5W speaker
  * 10,000 mAh power bank
* Used as:

  * Smart home controller
  * Portable voice assistant
* Controlled via **WearOS Alexa app**

## 🛏️ Room #2 – Smart AC + Lighting

### ⚙️ Components

* Philips Wiz Smart LED Bulb (10W)
* NodeMCU ESP8266 (Tasmota firmware)
* 40A Solid State Relay (SSR)
* DIY SSR (LED + LDR based)

### ❄️ AC Automation

* Runs until **5 AM**, then:

  * AC OFF
  * Stabilizer OFF after 5 minutes

### 💡 Lighting

* Dual-purpose:

  * Normal light
  * Night lamp (10% brightness)

### 🎛️ Control

* Mobile app
* Alexa / Google Assistant
* WearOS Alexa integration

## 🛋️ Living Room – Entertainment Automation

### ⚙️ Hardware

* Echo Dot (3rd Gen)
* NodeMCU ESP8266
* 4-channel relay
* 5V 1A adapter

### 📺 Connected Devices

* TV
* LED strip (TV backlight)
* 5.1 Soundbar system

### 🔗 Platform

* Sinric Pro (Alexa & Google Assistant integration)

### 🎬 Automation Scenes

#### ▶️ "Alexa, turn on TV"

* Turns ON TV

#### 🍿 "Alexa, movie time"

* TV ON (if OFF)
* Soundbar ON (HDMI ARC auto-connect)
* LED strip ON
* Wiz bulb OFF

#### ⏸️ "Alexa, movie break"

* Media paused
* Lights ON at 50%

# 🏍️ Bike Keyless Entry via IoT

### ⚙️ Hardware

* NodeMCU ESP8266
* 4-channel relay
* 12V to USB power module

### 🔗 Platform

* Sinric Pro

### 🔌 Controls

* Ignition
* Kill switch
* Self-start

### 🎮 Extra Features

* Horn & high beam (limited use)
* Alexa / Google Assistant control
* WearOS Alexa integration

### 🔑 Keyless Entry System

* RF module (from RC car)
* Auto unlock when nearby
* Auto lock beyond ~10 meters

### 🌐 Connectivity

* Mobile hotspot (same SSID as home Wi-Fi)
* Optional Bluetooth control

### 🤖 Bonus

* DIY Bluetooth helmet receiver
* Start bike hands-free (Iron Man style 😄)
